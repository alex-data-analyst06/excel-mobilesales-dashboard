Excel Dashboard Project - Mobile Store sales analysis
# Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers

## Project Objective
To analyze mobile retail sales performance across brands, cities, and salespersons using Excel dashboard techniques.

## Key Insights
- Apple generated highest revenue among brands.
- Bangalore contributed highest city-wise revenue.
- Meena was the top-performing salesperson.
- Sales trend showed overall growth with minor fluctuations.
